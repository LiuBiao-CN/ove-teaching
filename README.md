<div align="center">
  光学虚拟实验教学演示项目开发
</div>
<br>
<strong>摘 要</strong>  为了解决光学课程教学中所涉及到的干涉、衍射以及信息光学实验方面的原理比较抽象、不直观、学生难以理解以及由于试验条件的限制使得学生不能充分的进行相关实验的问题，本文基于Matlab GUI和Application Complier工具设计开发了一款基于Windows操作系统运行的光学虚拟实验教学演示平台软件，实现了对光学实验课程教学中常见的干涉、衍射以及全息、空间滤波等信息光学实验的模拟仿真。同时，该演示平台的图形用户界面友好、直观、交互性以及可移植性强，通过该演示平台的使用，能够提高学生的学习兴趣，帮助学生提前预习和更好地去理解相关的实验，同时也可以帮助教师在光学理论课教学过程中将抽象的理论与实验效果展示相结合，提高教学质量。<br>
<strong>关键词</strong>  干涉，衍射，信息光学，光学实验仿真，图形用户界面
